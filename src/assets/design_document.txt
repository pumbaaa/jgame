Package      Class                        Info

audioPlayer
(Static)      AudioPlayer                 Manage all the sound and music                 

background
              Background                  Class for making the backgrounds
             
gameState
              GameState                   Abstract SuperClass for the different
                                          states
              StateMainMenu               The main menu
              StateLevel1                 Level 1
              GameStateManager            Manage all the states through calls to
                                          this manager

main
              Game                        Contains the main game loop
              OOP2016_a15eriho_uppgift6   Starting point of the application

mapObject
              Animation                   To animate a map object
              MapObject                   Abstract super class for single map
                                          objects
              Player                      The player
              Bullet                      A bullet
              Mushroom                    An enemy

settings
              Settings                    Contains a set of constants
      
             
tileManager
(Static)      TileManager                 Manage tiles from the tile sheets
              Tile                        Contains a BufferedImage for a single
                                          tile

tileMap
              TileMap                     Abstract super class for the maps
              TileInfo                    The information for a specific tile on
                                          a map
